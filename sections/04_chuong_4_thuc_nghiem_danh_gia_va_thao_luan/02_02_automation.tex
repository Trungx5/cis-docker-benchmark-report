\noindent\textbf{Chi tiết kịch bản kiểm toán tự động (Ansible Playbook) cho Section 1 và Section 6:}

Dưới đây là chi tiết các tác vụ (tasks) trong playbook Ansible tự động được sử dụng để kiểm toán các khuyến nghị thuộc Section 1 (Host Configuration) và Section 6 (System Operations):

\noindent\textbf{Section 1 -- Host Configuration}

Kịch bản kiểm toán tự động các khuyến nghị từ 1.1.1 đến 1.1.18 và từ 1.2.1 đến 1.2.2:

# Kiểm toán 1.1.1: Đảm bảo phân vùng riêng cho container
- name: 1.1.1 Ensure a separate partition for containers has been created
  ansible.builtin.command:
    cmd: "docker info -f '{{ .DockerRootDir }}'"
  register: docker_info

- name: Check user namespace
  ansible.builtin.shell:
    cmd: "docker info | grep -q userns"
  failed_when: false
  register: userns_output

- name: Set path to check (userns enabled)
  ansible.builtin.set_fact:
    docker_root_path: "{{ docker_info.stdout }}/.."
  when: userns_output.rc == 0

- name: Set path to check (userns disabled)
  ansible.builtin.set_fact:
    docker_root_path: "{{ docker_info.stdout }}"
  when: userns_output.rc == 1

- name: Check mount point
  ansible.builtin.command:
    cmd: 'mountpoint -q -- "{{ docker_root_path }}"'
  register: mount_point
  failed_when: false

- name: Pass / Fail Check 1.1.1
  ansible.builtin.assert:
    that:
      - mount_point.rc == 0
    success_msg: "[PASS] 1.1.1 - Separate mount point created"
    fail_msg: "[FAIL] 1.1.1 - No separate mount point created"
  register: assert_1_1_1
  failed_when: false

# Kiểm toán 1.1.2: Trusted users kiểm soát Docker Daemon
- name: 1.1.2 Ensure only trusted users are allowed to control Docker daemon
  ansible.builtin.command:
    cmd: getent group docker
  register: docker_group

- name: Print users in docker group
  ansible.builtin.debug:
    msg: |
      [INFO] 1.1.2 - Users in docker group: {{ docker_group.stdout }}

# Kiểm toán 1.1.3: Cấu hình kiểm toán cho Docker daemon
- name: 1.1.3 Ensure auditing is configured for the Docker daemon
  ansible.builtin.shell:
    cmd: apt list --installed | grep -w auditd
  register: auditd_output
  failed_when: false

- name: Check auditing is configured

    cmd: auditctl -l | grep /usr/bin/dockerd
  register: auditing_1_1_3
  failed_when: false
  when: auditd_present

# Kiểm toán 1.1.4 - 1.1.6: Kiểm toán /run/containerd, /var/lib/docker, /etc/docker

- name: Check auditing is configured for /etc/docker

# Kiểm toán 1.1.7: Kiểm toán docker.service
- name: 1.1.7 Ensure auditing is configured for docker.service
  ansible.builtin.systemd_service:
    name: docker.service
  register: docker_service_command

- name: Set fact FragmentPath for docker.service

# Kiểm toán 1.1.8 - 1.1.12: Kiểm toán socket và cấu hình daemon
- name: 1.1.8 Ensure auditing is configured for containerd.sock
  ansible.builtin.stat:
    path: /run/containerd/containerd.sock
  register: containered_stats

- name: Check if auditing is enabled for containerd.sock
  ansible.builtin.shell:
    cmd: auditctl -l | grep /run/containerd/containerd.sock
  register: containerd_command
  when: auditd_present and containered_stats.stat.exists
  failed_when: false

- name: 1.1.9 Ensure auditing is configured for docker.socket
  ansible.builtin.systemd_service:
    name: docker.socket
  register: docker_socket_command

- name: Read docker.socket content
  ansible.builtin.slurp:
    src: "{{ docker_socket_command.status.FragmentPath }}"
  register: fragment_path_base64

- name: Look for socket path
  ansible.builtin.set_fact:
    docker_socket_path: >-
      {{ fragment_path_base64.content | b64decode
      | regex_search('^ListenStream(.+)', multiline=True)
      | regex_replace('ListenStream=','') | trim }}
  when: docker_socket_command.status.FragmentPath != ""

- name: Check docker.socket file exists
  ansible.builtin.stat:
    path: "{{ docker_socket_path }}"
  register: docker_socket_file
  when: docker_socket_command.status.FragmentPath != ""

- name: Ensure auditing is configured for docker.socket
  ansible.builtin.shell:
    cmd: "auditctl -l | grep {{ docker_socket_path }}"
  register: auditing_1_1_9
  when: auditd_present and docker_socket_file.stat.exists
  failed_when: false

    msg: "[INFO] 1.2.1 - Ensure the container host has been Hardened"

- name: 1.2.2 Ensure that the version of Docker is up to date
  ansible.builtin.apt:
    update_cache: true

- name: Check if new updates are available for docker
  ansible.builtin.command:
    cmd: apt list --upgradeable
  register: apt_list_command

- name: Assert new updates for docker
  ansible.builtin.assert:
    success_msg: "[FAIL] 1.2.2 - There are new updates for Docker"
    that:
      - apt_list_command.stdout is regex('^docker-ce/', multiline=True)
      - apt_list_command.stdout is regex('^docker-ce-cli/', multiline=True)
    fail_msg: "[PASS] 1.2.2 - There are no new updates for Docker"
  register: assert_1_2_2
  failed_when: false

\noindent\textbf{Section 6 -- Docker Security Operations}

Kịch bản kiểm toán tự động chống image sprawl và container sprawl:

{\footnotesize
\begin{verbatim}
# Kiểm toán 6.1: Tránh tích tụ quá nhiều image (image sprawl)
- name: "6.1 | Count images actively used by running containers"
  ansible.builtin.shell:
    cmd: |
      docker ps -qa | xargs -r docker inspect --format "{{.Image}}" | sort -u | wc -l
  register: active_images_command
  changed_when: false
  failed_when: false

- name: "6.1 | Count all images on host"
  ansible.builtin.shell:
    cmd: "docker image list -a -q | wc -l"
  register: all_images_command
  changed_when: false
  failed_when: false

# Kiểm toán 6.2: Tránh tích tụ quá nhiều container dừng (container sprawl)
- name: "6.2 | Get total container count"
  ansible.builtin.shell:
    cmd: "docker info --format '{{ .Containers }}'"
  register: total_container_command
  changed_when: false
  failed_when: false

- name: "6.2 | Get running container count"
  ansible.builtin.shell:
    cmd: "docker info --format '{{ .ContainersRunning }}'"
  register: total_running_container_command
  changed_when: false
  failed_when: false

- name: "6.2 | Get stopped container count"
  ansible.builtin.shell:
    cmd: "docker info --format '{{ .ContainersStopped }}'"
  register: total_not_running_container_command
  changed_when: false
  failed_when: false
\end{verbatim}
